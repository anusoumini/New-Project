# Movie Recommender System

## Description
This project recommends movies based on user preferences using machine learning.

## Dataset
Download dataset from:
movies dataset
https://drive.google.com/file/d/1ZnxZaDE-fPr5ynaPdcHSU_cJzNHBBWCl/view?usp=sharing

ratings dataset
https://drive.google.com/file/d/1DS6_p4OTds-LOV2hQDcWoErJCIbavqtj/view?usp=sharing

credits dataset
https://drive.google.com/file/d/18gwV3UZjmwbxnSTFljR_zFYEIcQBl8AK/view?usp=sharing

## Setup
1. Download dataset
2. place in same folder as main.py
3. Run:
   python main.py
